# LeoLibrary V1 Scope

LeoLibrary is a native audiobook library for Mac OS X Leopard.

## In Scope

- Manage local audiobook collections
- Import existing audiobook files and folders
- Store audiobook metadata
- Display cover artwork
- Search and filter the local library
- Reveal files in Finder
- Play supported local audio through Leopard-native facilities where possible
- Export library data as simple catalog files

## Out of Scope

- General media management
- Streaming service integration
- Audible account automation
- DRM circumvention
- Activation-byte handling
- Automated conversion of protected content
- Cross-platform UI layers
- Java runtime dependency
- SWT, Maven, or Install4J based application architecture
